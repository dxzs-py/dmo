## &#xe635; 项目经历

<div alt="entry-title">
    <h3>宁夏特色作物气象灾害预警系统</h3>
    <a href="https://github.com/dxzs-py/myxm/tree/master">github.com/dxzs-py/myxm</a>
</div>

➢ 项目描述

聚焦宁夏特色作物气象灾害防控痛点，融合深度学习、时序预测、智能判别等AI核心技术，联合宁夏气象科学研究所搭建气象灾害风险识别与预警系统。整合1961-2024年气象数据构建专属AI训练数据集，基于LSTM+Transformer混合AI模型实现关键气象因子精准预测、灾害等级智能量化判别，结合Django+Vue开发Web平台，打造AI全链路赋能体系，破解传统农业气象服务的技术短板，为宁夏枸杞、葡萄、小麦等特色作物提供智能化灾害防控支撑。

➢ 项目流程

1. AI专属数据集构建：获取宁夏气象、灾害及作物生长数据，完成清洗、归一化预处理，提取降水变化率等核心特征，通过14天一组步长1的方式数据增强，划分训练/测试集，构建标注完善的气象灾害数据集。
2. 混合AI模型训练：基于Pytorch搭建LSTM+Transformer模型，小批量梯度下降优化参数，引入Dropout防止过拟合，经多轮迭代训练，实现模型在测试集的高精度灾害识别与预测。
3. AI模型接口开发：按气象局标准构建灾害判别接口类，封装模型为后端API，实现实时气象数据输入后，模型自动判别灾害类型、划分灾害等级。
4. AI模型部署与优化：将训练完成的模型部署至后端，前后端集成实现AI推理结果可视化；经气象场景实测试验后调优模型参数，持续收集数据进行增量训练，提升模型实际适配性。
   ➢ 核心AI技术应用

1. LSTM时序预测：捕捉气象数据的时间序列特征，解决梯度消失问题，精准预测未来15天气温、湿度、降水量等关键气象因子。
2. Transformer注意力融合：让模型自动聚焦对作物生长影响的核心气象因子，提升关键信息提取能力，增强灾害预测的针对性。
3. AI智能灾害判级：构建端到端分类模型，替代人工经验判断，实现干旱、霜冻等灾害类型及等级的自动化、标准化判别。
4. AI特征工程与数据挖掘：从海量数据中筛选与作物灾害强相关的气象特征，构建最优特征集，提升模型推理效率与精度。
   ➢ 核心功能实现

1. AI气象预测可视化：展示实时气象数据，及AI模型预测的未来15天数据，以趋势图形式实现可视化对比分析。

2. AI智能灾害预警：模型实时分析实时数据，自动发布分级预警，展示预警详情、判别依据及针对性防护建议。

3. AI个性化种植服务：根据用户定制的关注区域/作物，推送专属的气象预测结果与灾害预警信息。

4. AI作物生长分析：AI分析实现作物分布、种植面积地图可视化，自动判断作物当前生长阶段并预测下一阶段进入时间。

    <h3>剧本编写智能体开发</h3> 
    <a href="https://github.com/YiNNx/cmd-wrapped">github.com/octocat/gitflix</a>

➢ 项目描述

聚焦剧本创作效率与创意挖掘需求，依托 LangChain v1.2.7框架搭建剧本编写智能体，融合 RAG（检索增强生成）技术，构建从创意挖掘到剧本生成的全流程服务。接入海量影视剧本库（涵盖剧情框架、人物设定等）、文学素材库及行业知识资源，实现剧本创意启发、情节自动生成与专业规范校验，助力编剧突破创作瓶颈，提升剧本产出质量与效率，探索 AIGC 在影视创作领域的深度应用。

➢ 项目流程

1. **LangChain 框架部署**
   基于 Python 环境配置 LangChain，设计剧本创作流程节点，包括创意输入、意图识别、检索增强、生成校验等，构建智能体工作流，保障流程自动化与可扩展。

2. **数据整合预处理**
   - 梳理影视剧本库（存储剧本分镜、人物台词等）、文学素材库（小说、散文等文本），通过 Sentence - BERT 模型转化为向量，设置向量维度为 768，存入 Milvus 向量数据库，支持语义检索，向量检索耗时控制在 200ms 内。
 - 用 MySQL 存储剧本元数据（作者、题材等）、行业规范（格式标准、审核要求）等结构化信息，与向量库协同，确保元数据查询响应时间 ≤ 50ms。

3. **用户交互与意图识别**
 用户输入剧本创作需求（如“悬疑短剧开头情节”“古装甜宠人物小传”），经基于 BERT 微调的意图识别模型处理，分类专业创作咨询、创意发散需求，模型分类准确率达 88%。
   - 任务类型：专业咨询（如剧本格式规范、行业审核要点）：触发直接检索流程，调用“知识问答模型（LLM + Prompt）”，从 MySQL 结构化知识库精准提取答案，答案提取准确率超 92%。

4. **RAG 驱动剧本生成**
 - **检索增强**：根据用户意图，从 Milvus 向量库召回 Top3 - Top5 相关爆款素材，通过 Prompt 工程注入大模型。
   - **剧本生成**：调用本地部署的 DeepSeek - v3 大模型，基于检索素材与创意需求，生成剧本片段（含人物对话、场景描述），单片段生成耗时 ≤ 1s。
   - **质量校验**：通过“剧本规范模型（LLM + Prompt）”，校验生成内容的格式合规性、情节逻辑性，校验覆盖率达 95%，输出校验报告与优化建议。

➢ 项目优化

  1. **模型微调与参数调优**
     针对本地部署的生成模型（如 DeepSeek 相关开源版本），基于剧本创作语料进行微调，设置学习率 2e-5、训练轮次 8 轮，重点优化剧情逻辑生成能力，使情节连贯性提升 25%；同时调整 Sentence - BERT 的向量生成参数，将批处理大小设为 32，提升检索向量与剧本场景的匹配度，素材相关性达 92%。
  
2. **模型蒸馏与量化压缩**
     采用知识蒸馏技术，将大模型（如 13B 参数版本）的创作能力迁移至 7B 轻量模型，保留核心逻辑生成能力的同时，模型体积缩减 45%。
