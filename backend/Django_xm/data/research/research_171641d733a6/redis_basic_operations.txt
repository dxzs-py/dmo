### Redis的基本操作

1. **添加数据**：
   - 使用 `SET key value` 命令可以向Redis中添加字符串数据。
   - 使用 `HSET key field value` 命令可以向哈希中添加数据。

2. **获取数据**：
   - 使用 `GET key` 命令可以获取指定key的字符串数据。
   - 使用 `HGET key field` 命令可以获取哈希中指定字段的值。

3. **删除数据**：
   - 使用 `DEL key` 命令可以删除指定的key及其对应的数据。

4. **修改数据**：
   - 使用 `SET key new_value` 命令可以修改字符串数据。
   - 使用 `HSET key field new_value` 可以修改哈希中指定字段的值。

5. **检查数据是否存在**：
   - 使用 `EXISTS key` 命令来判断指定的key是否存在，返回1表示存在，返回0表示不存在。

6. **数据类型转换**：
   - 使用 `TYPE key` 命令可以获取指定key的数据类型。

7. **事务操作**：
   - 使用 `MULTI` 开始一个事务。
   - 在事务中可以放置多个操作，最后使用 `EXEC` 提交事务，或者使用 `DISCARD` 取消事务。
